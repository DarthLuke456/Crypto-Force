TRADING AUTOMÁTICO CON BOT

Trading Automático

Rentabilidad dinámica de entre el 10% y 20% mensual. Funcionamiento las 24 horas, los 7 días de la semana.

REQUISITOS PARA USAR EL BOT

  • Solicite su link de registro único y particular de exchange.
  • Fondeo inicial en su cuenta de trading.
  • Creación de la API siguiendo las instrucciones proporcionadas.

PROCESO DE CONFIGURACIÓN

Cada usuario deberá tener su cuenta de exchange activa con el link único y particular que solicitará y su afiliado le compartirá.

Deberá realizar el proceso de creación de API explicado en el video compartido con tal fin, y compartir las claves y contraseña correspondientes para hacer el enlace con el servidor del servicio.

Cada usuario debe informar el monto que asignará al uso del bot, capital que debe estar en la wallet del exchange en el apartado Trading.

CÓMO ACTIVAR Y REACTIVAR EL BOT

  • Período inicial: Activación (60 días)
  • Período de renovación: Se comparte el 30% de las ganancias del período anterior.
  • Período siguiente: Se comparte el 30% de las ganancias del último período mensual.

Activación Inicial:

Desde su activación, comienza un período de 60 días. Al finalizar, el usuario deberá compartir el 30% de las ganancias para continuar con el uso.

Reactivación Mensual:

Al final de cada período mensual, el usuario deberá compartir el 30% de las ganancias para renovar su acceso al bot.

CONSIDERACIONES FINALES

  • Solo para uso del bot: El bot debe operar sin interferencia manual.
  • No trading manual: Es imperioso que en la cuenta donde se activa el bot no se realice trading manual, para no alterar los mandamientos del bot.
  • Ajuste de capital: Avisar antes de ingresar más capital.
  • Eventos y promociones: Se permite participar en promociones que impliquen transacciones de volumen.
  • Subcuenta de trading: Para operaciones manuales, se debe crear una subcuenta.
